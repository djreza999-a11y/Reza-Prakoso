# Website Portofolio Profesional - Reza Prakoso Dwi Julianto

Ini adalah draft awal website portofolio pribadi/profil profesional untuk Reza Prakoso Dwi Julianto, S.P., M.P.

## Cara membuka

Buka file `index.html` langsung di browser.

## Bagian yang perlu diganti

- Foto profil resmi, jika ingin ditambahkan
- Detail proyek penelitian pilihan
- Daftar publikasi unggulan
- Sertifikasi atau tautan akademik tambahan
- Tautan Google Scholar/SINTA/Scopus jika tersedia

## Publikasi gratis

Website ini bisa dipublikasikan gratis menggunakan GitHub Pages, Cloudflare Pages, atau Netlify.
